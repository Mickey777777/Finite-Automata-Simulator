# Finite Automata Simulator

브라우저에서 동작하는 **DFA(결정적 유한 오토마타) 시각화 및 시뮬레이션 도구**입니다.

상태, 알파벳, 전이 함수를 입력하면 오토마타 다이어그램을 SVG로 렌더링하고, 입력 문자열의 수락/거부 여부를 시뮬레이션합니다.

## 기능

- **DFA 정의** — 상태집합(Q), 알파벳(Σ), 시작 상태(q₀), 수락 상태(F), 전이 함수(δ)를 직접 입력
- **상태 다이어그램 시각화** — SVG 기반 그래프 자동 생성 (원형 배치)
- **문자열 시뮬레이션** — 입력 문자열에 대한 Accept/Reject 판정 및 경로 표시
- **시뮬레이션 경로 하이라이트** — 실행 경로의 간선 색상이 결과에 따라 변경 (Accept: 초록, Reject: 빨강)
- **드래그 & 줌** — 노드 드래그로 배치 조정, 캔버스 패닝 및 마우스 휠 줌 지원

## 사용법

1. `index.html`을 브라우저에서 엽니다.
2. 좌측 패널에서 DFA의 5-튜플을 입력합니다.
   - **Q**: `q0, q1, q2`
   - **Σ**: `0, 1`
   - **q₀**: `q0`
   - **F**: `q2`
3. 전이 함수 테이블에 각 상태-심볼 쌍의 다음 상태를 입력합니다.
4. **▶ DFA 시각화** 버튼을 클릭하면 다이어그램이 생성됩니다.
5. 입력 문자열(예: `0110`)을 넣고 **실행** 버튼으로 시뮬레이션합니다.
