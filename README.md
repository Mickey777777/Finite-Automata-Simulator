# Finite Automata Simulator

브라우저에서 동작하는 **DFA / NFA 시각화 및 시뮬레이션 도구**입니다.

상태, 알파벳, 전이 함수를 입력하면 오토마타 다이어그램을 SVG로 렌더링하고, 입력 문자열의 수락/거부 여부를 시뮬레이션합니다.

## 기능

### DFA (결정적 유한 오토마타)
- 상태집합(Q), 알파벳(Σ), 시작 상태(q₀), 수락 상태(F), 전이 함수(δ)를 직접 입력
- 입력 문자열에 대한 Accept/Reject 판정 및 경로 표시

### NFA (비결정적 유한 오토마타)
- λ-전이(빈 문자열 전이) 지원 — 전이 테이블에 λ 열 자동 추가
- 하나의 심볼에 대해 여러 상태로 전이 가능 (쉼표로 구분 입력)
- λ-closure 기반 시뮬레이션 — 각 스텝의 상태 집합을 추적하여 결과 표시

### 공통
- **상태 다이어그램 시각화** — SVG 기반 그래프 자동 생성 (원형 배치)
- **시뮬레이션 경로 하이라이트** — 실행 경로의 간선 색상이 결과에 따라 변경 (Accept: 초록, Reject: 빨강)
- **드래그 & 줌** — 노드 드래그로 배치 조정, 캔버스 패닝 및 마우스 휠 줌 지원

## 사용법

1. `index.html`을 브라우저에서 엽니다.
2. 헤더의 **DFA / NFA** 토글로 모드를 선택합니다.
3. 좌측 패널에서 오토마타의 5-튜플을 입력합니다.
   - **Q**: `q0, q1, q2`
   - **Σ**: `0, 1`
   - **q₀**: `q0`
   - **F**: `q2`
4. 전이 함수 테이블을 채웁니다.
   - DFA: 각 셀에 하나의 상태 (예: `q1`)
   - NFA: 쉼표로 여러 상태 입력 가능 (예: `q1, q2`), λ 열에는 빈 문자열 전이 입력
5. **시각화** 버튼을 클릭하면 다이어그램이 생성됩니다.
6. 입력 문자열(예: `0110`)을 넣고 **실행** 버튼으로 시뮬레이션합니다.

## 기술 스택

- **HTML / CSS / JavaScript** (순수 바닐라, 외부 라이브러리 없음)
- **SVG** — 상태 다이어그램 렌더링
